Projeto de Estrutura de Dados

Projeto de Estrutura de Dados para a matéria de SI201 (Estrutura de Dados 1) da UNICAMP. O projeto simula o funcionamento de uma loja, em que há uma fila de pedidos, cada pedido composto de uma lista de itens, e a quantidade de itens representada por uma pilha. Há ainda também uma lista ligada separada indicando o estoque da loja. O projeto foi inteiramente desenvolvido em C, com forte uso de ponteiros e structs.

Data Structures Project

Data Structures project for the SI201 (Data Structures 1) course at UNICAMP. The project simulates the operation of a store, where there is an order queue, each order consisting of a list of items, and the quantity of items represented by a stack. There is also a separate linked list indicating the store's inventory. The project was entirely developed in C, with extensive use of pointers and structs.
