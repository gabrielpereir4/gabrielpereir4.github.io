Algoritmo de Regressão Linear

Algoritmo de Regressão Linear Simples desenvolvido em Python, com as bibliotecas NumPy, SkLearn, Pandas e PyLab. O algoritmo lê um arquivo .CSV que contém informações sobre as configurações de motores de carros e suas respectivas emissões, e prevê as emissões para um grupo de teste. O algoritmo atinge um valor R-Quadrado de 0,75.

Linear Regression Algorithm

Simple Linear Regression Algorithm developed in Python using the NumPy, SkLearn, Pandas, and PyLab libraries. The algorithm reads a .CSV file containing information about car engine configurations and their respective emissions, and predicts emissions for a test group. The algorithm achieves an R-Squared value of 0.75.
