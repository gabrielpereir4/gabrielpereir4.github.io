Tradutor de Código Morse usando Arduino

Sistema montado com Arduino que lê a entrada de botões ligados a um Protoboard e traduz, baseado na duração do acionamento dos botões, o valor equivalente em código Morse. Código desenvolvido em C++.

Tecnologias Utilizadas

C++
Arduino
Eletrônica

Morse Code Translator using Arduino

System built with Arduino that reads button inputs connected to a breadboard and translates, based on the duration of the button presses, the equivalent value in Morse code. Code developed in C++.
